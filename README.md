# 📱 小红书封面生成器

一个完全免费的小红书封面在线制作工具，支持多种模板、实时预览、一键导出1080×1440标准尺寸PNG图片。

## ✨ 功能特点

- 🎨 **10+精美模板** - 涵盖美妆、穿搭、美食、知识、旅行等类型
- ⚡ **实时预览** - 所见即所得，编辑立即生效
- 🎯 **标准尺寸** - 1080×1440像素（3:4比例），完美适配小红书
- 🌈 **自定义配色** - 渐变背景、文字颜色、强调色随心调整
- 📝 **文字编辑** - 标题、副标题，支持字号调整
- 💾 **一键导出** - 高清PNG格式，直接上传使用
- 📱 **响应式设计** - 支持桌面和移动端访问
- 🆓 **完全免费** - 无需注册，无水印，无限制
